# Azure Database Administrator Associate

Administre una infraestructura de base de datos de SQL Server para bases de datos relacionales locales e hibridas en la nube mediante las ofertas de bases de datos relacionales PaaS de Microsoft.


## Comandos de preparación de ejercicio

Ejecute los siguientes comandos desde la shell integrada en el portal de Azure:

```bash
git clone  https://github.com/MicrosoftLearning/dp-300-database-administrator.git LabFiles
cd LabFiles/AllFiles/Labs/Setup
./deploy-sql-database.ps1
```
